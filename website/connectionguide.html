Automated Installation

1. Download Waloria's launcher and run it from a directory of your choice

2. Select your World of Warcraft 3.3.5a folder

3. Let the launcher install required patches, executables and AddOns

4. Click play!

Manual Installation

Important: All steps are compulsory.

1. Download patch-W.mpq and place it in your 3.3.5a client's \Data\ folder

2. Download AIO_client.rar, unrar and place it in your 3.3.5a client's addon folder

3. Download Wow.exe, unrar, and replace it with yours.

4. Change your realmlist to: set realmlist waloria.com

5. Delete cache and login!
